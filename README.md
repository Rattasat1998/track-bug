# Bug Tracking Web App

## Description
A bug tracking application built with React, Vite, and Firebase.

## Features
- **Bug Listing**: View bugs with priority, status, and assignee.
- **Add/Edit Bug**: Report new bugs or edit existing ones via a modal.
- **Inline Editing**: Update status, priority, title, and assignee directly from the list.
- **Real-time Updates**: Data is synced in real-time using Firestore.

## Tech Stack
- React
- Vite
- Firebase (Firestore)

## Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Configure Firebase in `.env`.
4. Run locally: `npm run dev`
